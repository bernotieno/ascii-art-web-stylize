About ASCII Art Web

Welcome to ASCII Art Web, an online tool that converts text into beautiful ASCII art! The project is an extension of a previous ASCII art generator, now enhanced with a web interface for easy access and use.

Features

  • Convert text into ASCII art using various banners such as 'standard', 'shadow', and 'thinkertoy'.
  • Simple and intuitive web interface for easy input and output of ASCII art.
  • Supports different styles and formats of ASCII art.
  • Displays error messages on a dedicated page for better user experience.

How It Works

To use the ASCII Art Web tool, simply navigate to the home page, enter your desired text, select a banner style, and click the "Generate" button. The generated ASCII art will be displayed on a different page. If there are any errors (e.g., unsupported characters), an appropriate error message will also be shown on a different page.
